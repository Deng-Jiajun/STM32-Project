# 51 MCU Project
> 51 单片机实验项目
## 文件结构
根目录下每一个文件夹代表一个项目，以 `LED` 为例：
```
├─LED 👈项目根目录
│  ├─Program 👈程序相关内容
│  │  ├─Project 👈Keil 项目
│  │  │  ├─Listings
│  │  │  └─Objects
│  │  └─Source 👈项目源代码
│  └─Stimulation 👈仿真项目（Proteus）
```
> ps: 生成的 `.hex` 文件在`\Program\Project\Objects`目录下
